# FCNCAna

This is a repo for FCNC analysis. 

# setup 

git clone https://github.com/shchauha/FCNCAna

cd FCNCAna

source setup.sh

cd analysis/yields

root -l -b doAll2016.C 

will produce histograms for various samples.

# plotting 

use "plot.py" script in yields to make stacked plots. 

run it as: 

python plot.py
