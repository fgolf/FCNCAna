# FCNCAna

This is a repo for FCNC analysis. 

# setup 

git clone https://github.com/shchauha/FCNCAna

cd FCNCAna

source setup.sh

cd analysis/yields

python doAll.py

will produce histograms for various samples.

# plotting 

use "plot_all.py" script in yields to make stacked plots. 

run it as: 

python plot_all.py
